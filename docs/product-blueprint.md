# 제품 기준 문서

## 한 문장 목표

사업 공고문과 제출 양식을 읽고, 저장된 좋은 사례와 사용자의 아이디어를 결합해, 지원사업 목적에 맞는 사업계획서를 함께 작성하는 코워크 에이전트.

## 사용자가 원하는 결과

- 공고문을 PDF, Word, 텍스트로 올리면 핵심 목적과 평가기준을 정리한다.
- 부처별 양식이 다르더라도 업로드한 양식의 구조에 맞춰 초안을 만든다.
- 사용자가 모아 둔 좋은 사례 약 30개를 참고 자료로 삼는다.
- 좋은 사례의 문장보다 구조, 논리, 평가자 신호를 가져온다.
- 혼자 쓰는 것보다 훨씬 빠르게 초안, 보완 질문, 성과지표, 실행계획을 만든다.
- 어려운 부분은 AI에게 복사해 물어볼 질문으로 만들어 준다.
- 로컬과 Google Drive에 동시에 보관해 자료가 쌓인다.

## 핵심 경험

1. 공고문 넣기
2. 제출 양식 넣기
3. 좋은 사례 모으기
4. 내 아이디어 답하기
5. 관련 사례 찾기
6. AI에게 물어볼 질문 만들기
7. 답변 붙여넣기
8. 양식에 맞는 초안 만들기
9. Google Drive와 로컬에 저장하기

## 무료/오픈소스로 가능한 영역

- 텍스트 추출: PDF.js, Mammoth.js, Python 문서 파서
- 사례 검색: MiniSearch, FlexSearch, Transformers.js, SQLite/DuckDB 계열
- 구조도/흐름도: Mermaid, Excalidraw, draw.io
- 로컬 저장: IndexedDB, JSON 백업
- Google Drive 백업: Google Apps Script, Drive API
- Google Sheets 생성: 서비스 계정, Sheets API, Drive API

## 외부 AI에게 맡기는 영역

- 공고문의 숨은 평가 의도 해석
- 좋은 사례의 기획 논리 분해
- 사용자의 아이디어를 정책 언어로 번역
- 문항별 설득력 있는 문장 작성
- 시각 자료 설명문과 구조도 초안 생성
- 최신 트렌드 검색과 근거 요약

## 오픈소스 시각화 방향

Napkin.ai 같은 자동 인포그래픽 경험을 바로 복제하기보다, 아래 순서로 간단하게 구현한다.

1. AI가 Mermaid 코드 또는 표 구조를 생성한다.
2. 앱이 Mermaid 다이어그램으로 미리 보여준다.
3. 필요하면 Excalidraw, draw.io, tldraw 쪽으로 내보낸다.
4. 최종 제출물에는 PNG/SVG로 붙인다.

후보:

- Mermaid: 텍스트에서 흐름도, 타임라인, 마인드맵 생성
- Excalidraw: 손그림 느낌의 다이어그램과 PNG/SVG 내보내기
- draw.io/diagrams.net: 제출용 구조도와 프로세스 도식화
- tldraw + Mermaid: 편집 가능한 도형 변환

## 저장 구조

### 로컬

- 현재 프로젝트 입력값
- 저장한 사례 텍스트
- 검색 결과
- AI 질문과 답변
- 초안

### Google Drive

- 원본 공고문
- 원본 사업계획서 사례
- 양식 파일
- 백업 JSON
- 최종 산출물

### Google Sheets

- 공고 해석표
- 평가기준표
- 사례 분해표
- 성과지표표
- 실행계획표

## 다음 개발 우선순위

1. 로컬 저장을 `localStorage`에서 IndexedDB로 전환
2. PDF/DOCX 텍스트 추출 기능 추가
3. 업로드한 양식의 문항 구조 추출
4. 좋은 사례 30개를 사례 카드로 관리
5. 현재 공고와 가장 맞는 사례 자동 추천
6. AI 질문을 문항별로 자동 생성
7. Mermaid 기반 인포그래픽 초안 생성
8. Google Drive 양방향 동기화
9. Google Sheets 자동 정리 강화
10. 제출 양식 DOCX로 내보내기

## 설계 원칙

- 사용자는 어려운 용어를 몰라도 된다.
- 화면은 항상 다음 행동을 알려준다.
- 사용자의 아이디어를 일반 문장으로 덮어쓰지 않는다.
- 좋은 사례는 베끼지 않고 구조만 가져온다.
- 무료 기능으로 가능한 일은 로컬과 오픈소스로 처리한다.
- 강한 판단이 필요한 일은 최신 AI에게 물어볼 질문으로 만든다.
- 원본 자료는 Google Drive에 쌓고, 작업 데이터는 앱에서 빠르게 꺼낸다.
