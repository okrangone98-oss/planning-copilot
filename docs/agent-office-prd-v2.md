# PRD v2 — AI Local Creator Operating System

## AI Local Creator Operating System

### 대훈 전용 AI 로컬기획 사무국

---

## 1. 제품 개요

### 제품명

AI Local Creator Operating System  
(AI 로컬기획 사무국)

### 제품 비전

혼자 일하는 로컬크리에이터·기획자·중간지원조직 활동가가 AI 에이전트들과 협업하여 다음 일을 수행할 수 있는 AI 운영체계를 구축한다.

- 사업기획
- 콘텐츠 제작
- 자료조사
- 문서작성
- 행정업무
- 자동보고
- 브랜딩

---

## 2. 핵심 문제 정의

현재 1인 기획자/창업자는 다음 문제를 겪는다.

### 문제

- 반복 문서작업 과다
- 사업기획과 실행 동시 수행
- 콘텐츠 생산 지속 어려움
- 행정·홍보·운영을 혼자 처리
- 자료 정리 및 기억 관리 어려움
- 이메일·보고·자료조사 누락
- 업무 우선순위 혼란

### 해결 방향

AI를 단순 챗봇이 아니라 다음 역할 기반 에이전트 조직으로 구성한다.

- 조사원
- 기획자
- 비서
- 콘텐츠 제작자
- 검토자
- 보고자

---

## 3. 제품 목표

### 단기 목표 (MVP)

하나의 명령으로 다음 산출물을 만들 수 있도록 한다.

- 자료조사
- 블로그 초안
- 사업기획 초안
- 메일 초안
- 보고서 생성

### 중기 목표

- 밤새 자동 작업
- 아침 자동 보고
- Google Drive 장기 기억
- GitHub 작업 기록
- Gmail 리포트 자동 생성

### 장기 목표

AI 로컬기획 사무국 구축.

- AI 팀장
- AI 실무자
- AI 기록관리
- AI 운영관리
- AI 콘텐츠팀

---

## 4. 주요 사용자

### Primary User

1인 로컬크리에이터

특징:

- 콘텐츠 생산
- 사업기획
- 행정
- 홍보
- 교육기획
- 커뮤니티 운영

을 동시에 수행한다.

### 사용자 예시

- 농촌활성화지원센터 활동가
- 로컬브랜드 운영자
- 지역 콘텐츠 제작자
- 공모사업 기획자
- 협동조합 실무자
- 소규모 창업자

---

## 5. 핵심 기능

### 5-1. AI 사무국 (Chief Agent)

역할:

- 사용자 명령 분석
- 업무 분배
- 결과 통합
- 우선순위 판단
- 최종 보고 생성

예시:

```text
문화건강배달 후속사업 기획해줘
```

처리 흐름:

- 리서치 에이전트 호출
- 콘텐츠 에이전트 호출
- 사업기획 에이전트 호출
- 검토 에이전트 호출
- 최종 보고서 생성

### 5-2. 리서치 에이전트

역할:

- 공모사업 조사
- 사례 조사
- 정책자료 조사
- 시장조사
- 트렌드 분석

출력:

- 요약 보고서
- 참고 링크
- 핵심 인사이트

### 5-3. 콘텐츠 에이전트

역할:

- 블로그 초안
- 인스타 글
- 릴스 스크립트
- 뉴스레터
- 카드뉴스 문구

출력:

- Markdown 콘텐츠
- SNS 초안

### 5-4. 사업기획 에이전트

역할:

- 사업계획서 초안
- 문제정의
- 성과지표
- 실행계획
- 예산구조

출력:

- 사업기획 초안
- 논리모형
- 실행계획서

### 5-5. 행정문서 에이전트

역할:

- 공문
- 결과보고서
- 회의록
- 주민안내문
- 공고문

출력:

- 행정 문체 기반 문서

### 5-6. 메일 에이전트

역할:

- 메일 초안
- 제안 메일
- 강사 섭외 메일
- 업무 협조 메일

주의:

- 자동 발송 금지
- 초안만 생성

### 5-7. 검토 에이전트

역할:

- 논리 검토
- 예산 누락 확인
- 일정 검토
- 문체 통일

## 12. 성공 지표

### 단기

- 문서작성 시간 50% 감소
- 블로그 생산량 증가
- 사업기획 속도 향상

### 중기

- 월간 자동 콘텐츠 생산
- 야간 자동 리포트 운영
- 반복업무 자동화

### 장기

- 1인 AI 기획회사 운영 가능
- 지역 기반 AI 사무국 구축
- 로컬 AI 운영모델 사례화

---

## 13. 철학

이 시스템의 목적은 “AI가 사람을 대체하는 것”이 아니라 “혼자 일하는 사람에게 팀을 제공하는 것”이다.

AI는 실무를 보조하고, 사용자는 방향성과 판단을 담당한다.

---

## 14. 최종 비전

AI Local Creator Operating System은 로컬크리에이터, 농촌활동가, 중간지원조직, 1인 창업자가 AI와 함께 일하는 운영체계를 만드는 프로젝트이다.

이는 단순 생산성 도구가 아니라 “AI와 협업하는 새로운 노동 구조”를 만드는 것을 목표로 한다.
