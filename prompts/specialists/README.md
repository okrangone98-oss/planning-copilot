# Specialist Prompts

이 디렉토리는 기존 외부 에이전트 폴더에서 공개 가능한 지식만 정리한 전문 프롬프트 모음이다. 원본 폴더의 `.env`, 계정정보, 자동 게시 스크립트, 개인 메모는 포함하지 않는다.

## 사용 방식

1. AI 사무국에서 `GPT 질문 패키지`를 생성한다.
2. 필요한 전문 영역의 `prompt.md`를 함께 복사한다.
3. 유료 GPT/Claude에 붙여넣어 결과를 고도화한다.
4. 사실관계와 공개 가능 여부를 사람이 확인한 뒤 앱에 반영한다.

## 현재 스페셜리스트

- `instagram-content`: 인스타그램 카드뉴스, 릴스, 캡션, 콘텐츠 캘린더
- `design-director`: 브랜드 톤, 레이아웃, 카드뉴스 디자인 검수
- `dev-lead`: 개발 요구사항 분해, 자동화 설계, PR 검토
- `security-review`: 민감정보, 시크릿, 자동 발송 위험 점검
