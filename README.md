# 공동기획형 사업계획서 코파일럿

공고문을 해석하고, 사용자의 아이디어를 문제정의/논리모형/성과지표/실행계획/서비스 블루프린트/사업계획서 초안으로 발전시키는 로컬 웹앱입니다.

## 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 됩니다.

## 현재 기능

- 공고문 핵심 요구사항 정리
- 공고문 붙여넣기 기반 자동 분석
- 코워킹 질문 및 작성 전략 생성
- GPT 협업용 질문 생성 및 답변 반영 메모화
- RAG 지식방: 좋은 사업계획서 사례, 유사 공고, 평가표 저장
- 저장 문서 기반 로컬 벡터 검색 및 GPT 질문에 참고 조각 삽입
- 공고문 텍스트 파일 첨부 및 분석
- Google Drive 백업용 아카이브 내보내기/가져오기
- Google Apps Script Web App을 통한 Drive 백업
- 사용자 아이디어 인터뷰
- 문제정의 구조화
- 논리모형 설계
- 성과지표 설계
- 실행계획 설계
- 서비스 블루프린트 작성
- 사업계획서 초안 자동 조립
- 브라우저 로컬 저장
- Markdown 내보내기

## 권장 사용 흐름

1. `공고 해석` 탭에 공고문 핵심 내용을 붙여넣습니다.
2. `공고문 분석`을 눌러 정책 목적, 평가 포인트, 필수 조건을 자동으로 채웁니다.
3. `코워킹` 탭에서 에이전트 질문에 답하며 빠진 근거와 작성 전략을 보완합니다.
4. `RAG 지식방`에 좋은 사업계획서, 유사 공고, 평가표를 저장하고 `관련 사례 찾기`를 누릅니다.
5. `GPT 협업` 탭에서 질문을 생성하고 GPT 답변을 붙여넣어 반영 메모를 만듭니다.
6. `Drive 백업` 탭에서 아카이브를 다운로드하거나 Google Drive에 백업합니다.
7. `샘플 분해` 탭에 좋은 사업계획서나 유사 공고의 구조를 정리합니다.
8. 아이디어, 문제정의, 논리모형, 성과지표, 실행설계를 채운 뒤 `초안 조립`에서 초안을 생성합니다.

## 현재 아카이빙 구조

- 기본 저장: 브라우저 `localStorage`
- 저장 대상: 프로젝트 입력값, RAG 지식방 문서, GPT 협업 메모, 초안
- 한계: 같은 브라우저/기기 안에서만 유지되며 대용량 원본 파일 저장에는 적합하지 않음
- 백업: `Drive 백업` 탭에서 JSON 아카이브를 다운로드하거나 Google Apps Script를 통해 Google Drive에 저장

## Google Drive 백업 설정

1. [Google Apps Script](https://script.google.com/)에서 새 프로젝트를 만듭니다.
2. 앱의 `Drive 백업` 탭에 있는 Apps Script 코드 템플릿을 복사해 붙여넣습니다.
3. `배포` → `새 배포` → `웹 앱`으로 배포합니다.
4. 실행 권한은 `나`, 접근 권한은 개인 사용 목적에 맞게 설정합니다.
5. 발급된 Web App URL을 앱의 `Google Apps Script Web App URL` 칸에 붙여넣습니다.
6. `Drive에 백업`을 누르면 내 Google Drive의 `Planning Copilot Archive` 폴더에 JSON 아카이브가 저장됩니다.

## 하이브리드 설계 원칙

- 오픈소스/로컬 기능: 문서 파싱, 키워드 추출, 배점/마감/제출서류 탐지, 체크리스트, 저장, 내보내기
- GPT 협업 기능: 평가자 관점 해석, 문제정의 고도화, 아이디어 번역, 설득 문장 생성, 보완 질문 생성

## 추천 오픈소스 확장 후보

- MiniSearch/FlexSearch: 브라우저 내 빠른 전문 검색
- Transformers.js: 브라우저 로컬 임베딩 기반 의미 검색
- Mammoth.js: DOCX 사업계획서 샘플 텍스트 추출
- PDF.js: PDF 공고문/사업계획서 텍스트 추출

## 추천 MCP 확장 후보

- `document-parser-mcp`: HWPX, DOCX, PDF, TXT를 텍스트와 표 구조로 추출
- `grant-notice-mcp`: 소상공인24, K-Startup, 기업마당 등 공고 검색/수집
- `planning-rag-mcp`: 좋은 사업계획서 샘플을 벡터DB에 저장하고 유사 사례 검색
- `evaluation-rubric-mcp`: 공고문 평가표를 배점 기반 체크리스트로 변환
- `docx-export-mcp`: 완성 초안을 제출 양식 DOCX/HWPX에 맞춰 내보내기
