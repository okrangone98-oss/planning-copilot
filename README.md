# 공동기획형 사업계획서 도우미

공고문을 넣고, 좋은 사례를 모으고, AI에게 물어볼 질문을 만든 뒤 사업계획서 초안까지 정리하는 웹앱입니다.

## 실행 방법

브라우저에서 `index.html`을 열거나, GitHub Pages 주소로 접속하면 됩니다.

배포 주소:

https://okrangone98-oss.github.io/planning-copilot/

## 할 수 있는 일

- 공고문 붙여넣기 또는 파일 첨부
- 공고문에서 중요한 내용 정리
- 다음에 답해야 할 질문 만들기
- 좋은 사업계획서 사례 저장
- 저장한 사례에서 현재 사업에 맞는 참고 내용 찾기
- AI에게 복사해 물어볼 질문 만들기
- AI 답변을 붙여넣어 초안에 반영
- Google Drive 백업
- 사업계획서 초안 만들기

## 권장 사용 순서

1. `공고 해석`에 공고문을 넣고 `공고문 분석`을 누릅니다.
2. `사례 모으기`에 좋은 사업계획서나 비슷한 공고를 저장합니다.
3. `관련 사례 찾기`를 눌러 참고 내용을 찾습니다.
4. `AI 질문`에서 질문을 만들고 복사합니다.
5. AI 답변을 붙여넣고 `답변 반영 메모화`를 누릅니다.
6. `질문 받기`, `문제정의`, `성과지표`, `실행설계`를 채웁니다.
7. `초안 조립`에서 초안을 생성합니다.
8. `백업`에서 파일로 저장하거나 Google Drive에 백업합니다.

## 저장 방식

기본 저장은 사용 중인 브라우저에 됩니다. 다른 컴퓨터에서도 이어서 쓰려면 `백업` 탭에서 백업 파일을 내려받거나 Google Drive 백업을 설정하세요.

Google Drive 백업을 쓰려면 `백업` 탭의 연결 코드를 복사해 Google Apps Script에 붙여넣고, 발급된 연결 주소를 앱에 넣으면 됩니다.

## 개발자 참고

향후 확장 후보:

- 문서 읽기: PDF, DOCX, HWPX 파일에서 텍스트 추출
- 사례 검색 고도화: MiniSearch, FlexSearch, Transformers.js
- Google Drive 연동 강화: 원본 파일 업로드와 폴더별 정리
- 외부 도구 연결: 공고문 수집, 문서 변환, 제출 양식 내보내기
