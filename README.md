# 공동기획형 사업계획서 코파일럿

공고문을 해석하고, 사용자의 아이디어를 문제정의/논리모형/성과지표/실행계획/서비스 블루프린트/사업계획서 초안으로 발전시키는 로컬 웹앱입니다.

## 실행 방법

별도 설치 없이 `index.html` 파일을 브라우저에서 열면 됩니다.

## 현재 기능

- 공고문 핵심 요구사항 정리
- 공고문 붙여넣기 기반 자동 분석
- 코워킹 질문 및 작성 전략 생성
- GPT 협업용 질문 생성 및 답변 반영 메모화
- 사용자 아이디어 인터뷰
- 문제정의 구조화
- 논리모형 설계
- 성과지표 설계
- 실행계획 설계
- 서비스 블루프린트 작성
- 사업계획서 초안 자동 조립
- 브라우저 로컬 저장
- Markdown 내보내기

## 권장 사용 흐름

1. `공고 해석` 탭에 공고문 핵심 내용을 붙여넣습니다.
2. `공고문 분석`을 눌러 정책 목적, 평가 포인트, 필수 조건을 자동으로 채웁니다.
3. `코워킹` 탭에서 에이전트 질문에 답하며 빠진 근거와 작성 전략을 보완합니다.
4. `GPT 협업` 탭에서 질문을 생성하고 GPT 답변을 붙여넣어 반영 메모를 만듭니다.
5. `샘플 분해` 탭에 좋은 사업계획서나 유사 공고의 구조를 정리합니다.
6. 아이디어, 문제정의, 논리모형, 성과지표, 실행설계를 채운 뒤 `초안 조립`에서 초안을 생성합니다.

## 하이브리드 설계 원칙

- 오픈소스/로컬 기능: 문서 파싱, 키워드 추출, 배점/마감/제출서류 탐지, 체크리스트, 저장, 내보내기
- GPT 협업 기능: 평가자 관점 해석, 문제정의 고도화, 아이디어 번역, 설득 문장 생성, 보완 질문 생성

## 개발 방향

향후 GitHub 저장소와 연결한 뒤 다음 기능을 확장합니다.

- 좋은 사업계획서 샘플 분해 모듈
- 공고문 파일 업로드 및 자동 항목 추출
- 평가기준 기반 점검표
- AI API 연동을 통한 문장 개선 및 기획 피드백
- 사업 유형별 템플릿 라이브러리
